##修改个人信息

	(1)改昵称
		弹出数据框拿到数据
		发起请求: changeNicknameUrl
		4个参数: 两个是默认, 一个是userId,一个是新昵称
		
	(2)修改密码
		界面设置弹出div, 三个输入框(旧密码.新密码.重复密码)
		
		点击保存: 发起请求changePasswordUrl
		额外两个参数: 新密码和原密码
